상품·수익성 분석

제출 기준

대표 보고 전에 반드시 닫아야 할 항목

보기 좋은 문서보다 실제 제품과 계산 근거, 고객 언어가 먼저입니다. 빠진 항목은 가능한 자료 회수 경로를 끝까지 확인합니다.

검토 항목필수 내용완료 기준
고객 VOC만족, 불만, 반복 키워드와 대표 리뷰 근거제품 요구로 연결
성공 원인제품력, 가격, 채널, 마케팅, 리뷰 축적 효과를 분리하나의 원인으로 단정하지 않음
구매 페르소나연령, 구매자, 구매 목적, 사용 장면, 불안, 증거 요구신생아·토들러 구분

완료 선언 원칙

필수 자료가 없으면 먼저 다른 회수 경로를 사용합니다. 단순히 수집 실패라고 적고 끝내지 않습니다.

  • 공식몰 → 브랜드 판매처 → 주요 리테일 → 검색 결과 → 기존 검증 자산 순으로 회수
  • 실제품 보고서는 판매 링크와 확인일이 없으면 최종본으로 부르지 않음
  • 수치가 추정이면 실제 매출과 구분하고 가정의 영향을 밝힘
  • 전략팀은 결론을, 티처팀은 재현성과 설명 품질을 최종 점검
